Quick Sort Algorithm

Start Learning

Introduction

QuickSort is a sorting algorithm based on the Divide and Conquer technique. It picks an element as a pivot and partitions the given array around the picked pivot by placing the pivot in its correct position in the sorted array.

Algorithm Steps

  • Choose a Pivot: Select an element from the array as the pivot. The choice of pivot can vary (e.g., first element, last element, random element, or median).
  • Partition the Array: Rearrange the array around the pivot. After partitioning, all elements smaller than the pivot will be on its left, and all elements greater than the pivot will be on its right. The pivot is then in its correct position, and we obtain the index of the pivot.
  • Recursively Call: Recursively apply the same process to the two partitioned sub-arrays (left and right of the pivot).
  • Further divide these two subarrays into further halves, until the atomic (single element) subarray is reached, and further division is not possible.
  • Start combining the elements again using the merge process.
  • Base Case: The recursion stops when there is only one element left in the sub-array, as a single element is already sorted.

Quick Sort Code (C++)


// C++ program to implement Quick Sort
#include <bits/stdc++.h>
using namespace std;

int partition(vector<int>& arr, int low, int high) {
    int pivot = arr[high];
    int i = low - 1;

    for (int j = low; j <= high - 1; j++) {
        if (arr[j] < pivot) {
            i++;
            swap(arr[i], arr[j]);
        }
    }
    swap(arr[i + 1], arr[high]);
    return i + 1;
}

void quickSort(vector<int>& arr, int low, int high) {
    if (low < high) {
        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}

int main() {
    vector<int> arr = {10, 7, 8, 9, 1, 5};
    int n = arr.size();
    quickSort(arr, 0, n - 1);

    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    return 0;
}

Output

Sorted vector : 1 5 7 8 9 10

Complexity Analysis

  • Best Case: Ω(n log n), occurs when the pivot element divides the array into two equal halves.
  • Average Case: θ(n log n), occurs when the pivot divides the array into two parts, but not necessarily equal.
  • Worst Case: O(n²), occurs when the smallest or largest element is always chosen as the pivot (e.g., sorted arrays).
  • Auxiliary Space: O(n), due to the recursive call stack.
